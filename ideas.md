# Yuchani Tools Pro - 설계 및 개선 계획

## 프로젝트 개요
기존 "유차니 도구" 사이트를 글로벌 수익형 웹툴 플랫폼으로 확장합니다.
- **타겟**: 한국 포함 전세계 사용자
- **수익 모델**: 구글 애드센스, 프리미엄 기능
- **기술**: 정적 HTML/React 기반 (초기), 필요시 풀스택 확장

## 설계 철학 선택

### 선택된 접근법: "Modern Professional Minimalism"

**설계 이동**: 미니멀리즘 + 프로페셔널 + 글로벌 SaaS 스타일

**핵심 원칙**:
1. **명확한 정보 계층**: 사용자가 원하는 도구를 3초 내에 찾을 수 있어야 함
2. **신뢰성과 안정성**: 기업용 도구처럼 보이되 친근함 유지
3. **글로벌 접근성**: 다국어, 다양한 시간대, 모든 기기 지원
4. **성능 우선**: 빠른 로딩, 즉시 반응하는 인터페이스

**색상 철학**:
- **주색**: 진한 파란색 (`#1E40AF`) - 신뢰, 전문성
- **보조색**: 밝은 청록색 (`#06B6D4`) - 활동성, 현대성
- **배경**: 깨끗한 흰색 + 미세한 그레이 그라데이션
- **강조**: 주황색 (`#EA580C`) - 중요한 행동 유도 (CTA)
- **감정**: 신뢰감 있으면서도 접근 가능한 느낌

**레이아웃 패러다임**:
- 상단 고정 네비게이션 (언어 선택, 로고, 검색)
- 카테고리별 도구 그리드 (2열 모바일, 3열 태블릿, 4열 데스크톱)
- 하단 푸터 (리소스, 언어 선택, 소셜, 법률)
- 광고 배너는 사이드바 또는 콘텐츠 사이에 자연스럽게 배치

**시그니처 요소**:
1. **도구 카드**: 아이콘 + 제목 + 설명 + 태그 (카테고리/인기도/난이도)
2. **검색 바**: 실시간 필터링 + 인기 검색어 제안
3. **언어 전환기**: 헤더 우측 상단 + 푸터 (중복 제공)

**상호작용 철학**:
- 호버 시 카드 살짝 상승 (lift effect)
- 클릭 시 부드러운 페이드 인/아웃
- 로딩 상태 명확한 스켈레톤 UI
- 성공/에러 토스트 알림

**애니메이션**:
- 페이지 전환: 200ms 페이드
- 카드 호버: 150ms 스케일 + 섀도우
- 검색 결과: 100ms 슬라이드 인
- 언어 전환: 300ms 크로스페이드

**타이포그래피**:
- **헤더**: `Pretendard` 또는 `Inter` (한글 지원) - 700 weight
- **본문**: `Pretendard` 또는 `Inter` - 400/500 weight
- **강조**: 600 weight로 계층 구분

## 기술 구조

### 다국어 지원
- 언어: 한국어(ko), 영어(en), 일본어(ja), 중국어 간체(zh), 스페인어(es)
- 방식: URL 기반 라우팅 (`/ko/`, `/en/` 등) 또는 쿼리 파라미터 (`?lang=ko`)
- 저장소: React Context + localStorage (사용자 선택 기억)

### SEO 최적화
- 각 페이지 메타태그 (제목, 설명, OG 이미지)
- 구조화 데이터 (JSON-LD): WebSite, WebApplication, FAQPage
- Sitemap: 모든 언어 + 도구별 페이지 포함
- robots.txt: 크롤링 가이드
- hreflang 태그: 다국어 버전 연결

### 도구 추가 계획
**신규 도구** (글로벌 수요 높음):
1. YouTube 썸네일 다운로더
2. 이미지 WebP 변환기
3. 비밀번호 생성기 (보안 강화)
4. 색상 코드 변환기 (HEX ↔ RGB ↔ HSL)
5. 텍스트 음성 변환 (TTS)
6. 마크다운 미리보기
7. JSON 포매터 & 검증
8. 정규식 테스터

### 수익화 전략
1. **구글 애드센스**: 헤더 배너 + 사이드바 + 콘텐츠 중간
2. **프리미엄 기능**: 배치 처리, 고급 옵션, 광고 제거
3. **제휴 링크**: 관련 도구/서비스 추천
