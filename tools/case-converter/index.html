사용설명서 & FAQ

실무에서 바로 활용할 수 있도록 주요 질문과 절차를 정리했습니다.

1. 텍스트 입력

붙여넣기 또는 파일에서 복사한 텍스트를 상단 입력창에 넣습니다. 자동 변환이 켜져 있다면 즉시 결과가 나타납니다.

2. 정리 옵션 선택

공백 정리, 악센트 제거, 줄바꿈 유지 여부를 선택해 출력 규칙을 맞춥니다. 팀 컨벤션에 따라 기본값을 저장해 두세요.

3. 케이스 확인 & 복사

각 카드의 복사 버튼으로 원하는 케이스를 즉시 복사합니다. 키보드 단축키(Ctrl/⌘+Enter)로 변환을 재실행할 수 있습니다.

4. FAQ

  • 문장부호가 섞여 있어도 되나요? — 문장 케이스가 자동으로 감지합니다.
  • 확장자는 어떻게 처리되나요? — 파일 슬러그는 마지막 점(.) 이후 문자열을 확장자로 간주해 보존합니다.
  • 브라우저에 저장되나요? — 설정은 로컬 저장소에 저장됩니다.
