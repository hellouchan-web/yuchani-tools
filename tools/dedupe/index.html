도움말 & 팁
  • 엑셀에서 붙여넣을 때 탭 구분이 자동으로 유지되며, 필요한 경우 CSV로 내려받아 다른 툴에 연결하세요.
  • 이메일 비교 모드는 행 내 첫 번째 이메일 주소를 기준으로 중복을 판단합니다.
  • 정리 결과는 오른쪽 창에서 바로 확인하고, '결과 복사'로 CRM/메일러에 즉시 붙여넣을 수 있습니다.
  • 대량 데이터(수만 행)도 클라이언트에서 빠르게 처리하도록 브라우저 성능에 맞춰 최적화했습니다.
  • 작업 이력을 남기고 싶다면 아래 처리 로그를 복사해 검수 기록으로 활용하세요.
사용설명서 & FAQ
  1. 왼쪽 입력창에 명단을 붙여넣습니다.
  2. 중복 기준과 옵션(공백 제거, 대소문자 구분 등)을 선택합니다.
  3. 중복 제거 실행을 눌러 정리된 결과를 확인합니다.
  4. 필요 시 결과를 복사하거나 TXT로 다운로드합니다.
  5. 동일한 설정으로 계속 작업하려면 입력 시 자동 실행을 켜두세요.
