HTML 미리보기

브라우저에서 HTML을 작성하면 즉시 오른쪽 미리보기에서 결과를 확인합니다. 자동저장, 샘플 삽입, 다운로드/복사까지 한번에.

입력
저장됨
미리보기

노트: 기본적으로 스크립트는 차단됩니다. 필요한 경우 "스크립트 실행 허용"을 활성화하세요.

도움말 & 팁
  • 보안상 기본은 스크립트 차단입니다. 외부 리소스가 필요한 경우에만 허용을 켜세요.
  • OG/Twitter 카드 미리보기는 별도 툴에서 확인이 더 정확합니다.
  • 입력 내용은 브라우저 localStorage에 자동 저장됩니다.
사용설명서 · FAQ
HTML 미리보기 결과가 안 나와요

편집창 내용이 비어있지 않은지 확인하고, 우측 상단 새 탭으로 열기로도 테스트하세요. 보안상 스크립트는 기본 차단됩니다.

스크립트 실행은 기본 차단인가요?

네. iframe 샌드박스로 스크립트를 막습니다. 필요할 때만 상단 옵션에서 allow-scripts를 켜세요.

파일은 어디에 저장되나요?

입력 내용은 브라우저 localStorage에만 저장됩니다. 서버로 전송하지 않습니다.
