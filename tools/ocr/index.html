도움말 & 팁
  • 짧은 변 기준 1200px 이상의 선명한 이미지를 사용하면 정확도가 크게 향상됩니다.
  • 기울어진 스캔본은 먼저 회전·자르기로 정렬하고, 그림자는 제거하세요.
  • 다국어 문서는 필요한 언어를 모두 선택하세요. (예: 한국어+영어)
  • 개인정보가 포함된 텍스트는 복사·공유 전에 한번 더 검토하세요.
  • 한 번 로드된 언어 데이터는 브라우저 캐시에 저장되어 이후 더 빠르게 동작합니다.
사용설명서 & FAQ
  1. 이미지 가져오기: 파일 선택, 드래그앤드롭, 붙여넣기 모두 지원합니다.
  2. 언어·모드 선택: 기본은 빠른 모드이며, 정밀도가 필요하면 ‘정확도 우선’을 켜세요.
  3. OCR 시작: 진행 상황과 단계가 표시되며 완료되면 결과가 오른쪽 상자에 나타납니다.
  4. 결과 활용: 복사 버튼 또는 텍스트 파일로 다운로드하여 문서 편집에 바로 사용하세요.
  5. 문제 해결: 네트워크 차단으로 실패하면 안내 메시지를 확인하고, 언어 데이터를 직접 호스팅하여 사용할 수 있습니다.
