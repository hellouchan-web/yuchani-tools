QR 코드 생성기

텍스트 또는 URL을 입력하고 옵션을 조정하면 즉시 오른쪽 미리보기로 확인하고 다운로드할 수 있습니다. 인쇄용은 SVG 또는 큰 픽셀 크기를 권장합니다.

입력
전경/배경은 충분한 대비를 권장
완료
미리보기
SVG 권장 (인쇄/확대 선명)
사용설명서
  1. 텍스트/URL 입력: 상단 입력란에 링크나 텍스트를 적습니다. 예) https://yuchani.com
  2. 크기·여백·오류복원(ECC): 인쇄용은 480px+ & SVG 추천. 로고를 올릴 계획이면 ECC를 Q/H로.
  3. 색상 대비: 전경은 진하게(#000), 배경은 밝게(#FFF). 대비가 낮으면 스캔 실패합니다.
  4. 미리보기 배경: 다크/라이트/투명 중 선택해 실제 사용 환경과 비슷하게 확인.
  5. 저장/공유: 이미지 URL 복사로 링크 공유하거나 이미지 다운로드로 PNG/SVG 저장.

팁: URL이 길면 먼저 단축해서 넣으면 스캔 성공률이 올라갑니다.

빠른 체크
  • ✅ 인쇄용이면 SVG 선택
  • ✅ 대비 충분? (검정 전경 · 흰 배경)
  • ✅ 여백(qzone) 4 이상 권장
  • ✅ 로고 올릴 땐 ECC Q/H
도움말 & 팁
  • 명함/전단지 인쇄: 흑백 대비가 가장 안정적. 색상 사용 시 충분한 대비를 확보하세요.
  • URL이 너무 길면 URL 단축 후 생성하면 스캔 성공률이 올라갑니다.
  • 오류복원(Q/H)을 올리면 로고 오버레이 같은 꾸밈에 유리하지만, 데이터 용량이 증가합니다.
