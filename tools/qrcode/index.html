QR 코드 생성기

텍스트·URL 입력 즉시 QR 코드를 생성하고, 크기·여백·오류복원·색상까지 전문가 기준으로 조정할 수 있는 고급 웹툴입니다. PNG/SVG 다운로드와 URL 복사를 모두 지원합니다.

PNG·SVG 동시 지원
대비·오류복원 가이드 내장
100% 브라우저 처리 · 개인정보 저장 없음

입력 & 옵션

전경은 진하게, 배경은 밝게 유지하세요.

주소창에 기록되지 않으며, 입력값은 브라우저에서만 처리됩니다.

대비 팁: 전경 대비가 4.5:1 이상이면 모바일 카메라에서 안정적으로 스캔됩니다.

실시간 미리보기

SVG는 로고 겹침·인쇄용에 최적화

스캔 가이드
여백(QZ)은 4px 이상 유지하면 카메라가 모듈을 빠르게 감지합니다.
색상 대비
전경이 배경보다 최소 40% 이상 진해야 대부분의 리더 앱에서 인식됩니다.
보안
API 호출은 암호화된 HTTPS로 전송되며, 어떤 데이터도 저장하지 않습니다.

사용 설명서

  1. 입력: 링크나 텍스트를 붙여넣으면 0.2초 디바운스로 자동 생성됩니다.
  2. 크기·여백: 인쇄/프린트용은 480px 이상, 여백은 4~8px을 추천합니다.
  3. 색상 대비: 전경은 진한 색(#000 등), 배경은 밝은 색(#FFF 등)으로 설정하세요.
  4. 미리보기 배경: 다크/라이트/투명 옵션으로 실제 환경에 맞춰 시뮬레이션합니다.
  5. 다운로드: PNG는 즉시 사용, SVG는 편집·인쇄용으로 적합합니다.

팁: 긴 URL은 단축한 뒤 삽입하면 패턴이 단순해져 스캔 성공률이 높아집니다.

빠른 체크리스트

  • ✅ 인쇄용이면 SVG 선택
  • ✅ 대비 충분? (진한 전경 · 밝은 배경)
  • ✅ 여백(QZ) 최소 4px
  • ✅ 로고 덮을 땐 ECC Q/H

도움말 & 팁

  • 명함·전단지 인쇄 시에는 CMYK 대비도 고려해 검정/흰색 조합을 권장합니다.
  • 웹사이트에 삽입할 경우 .svg를 사용하면 레티나 디스플레이에서도 픽셀이 깨지지 않습니다.
  • 오류 복원을 올리면 패턴이 복잡해져 용량이 증가하므로, 너무 긴 텍스트는 URL 단축 후 사용하세요.

FAQ

QR 코드는 어디에서 생성되나요?

공개된 api.qrserver.com 이미지를 호출하며, HTTPS 기반이라 안전합니다.

QR을 교체해도 링크는 유지되나요?

QR은 정적 이미지이므로 이미 배포한 이미지를 교체하면 새로운 링크로 업데이트되지 않습니다. 다이나믹 QR 서비스가 필요한 경우 별도의 리다이렉트 서버가 필요합니다.

모바일에서 저장이 안 될 때는?

iOS Safari 등에서는 팝업 차단이 원인일 수 있습니다. 공유 버튼을 눌러 '이미지 저장'을 선택하세요.
