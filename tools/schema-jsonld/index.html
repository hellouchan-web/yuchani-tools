FAQ 항목

FAQ는 2~8개가 이상적이며, 실제 페이지에 표시되는 질문/답변과 동일해야 합니다.

JSON-LD 출력
구글 리치결과 테스트
SEO 체크리스트
  • 실제 페이지에 동일한 콘텐츠(별점, FAQ)를 노출하세요.
  • FAQ는 사용자 질문에 답변하는 형식으로 작성합니다.
  • 평점은 신뢰 가능한 출처에서 수집한 실제 데이터를 사용하세요.
  • 추가 마크업(Organization, Breadcrumb 등)을 함께 사용하면 더 좋습니다.
  • 적용 후 구글 서치콘솔에서 색인 상태를 확인하세요.
도움말 & 팁
  • 페이지 타입에 따라 스키마 타입을 선택하고 세부 필드를 조정하세요.
  • 발행일이 있는 콘텐츠는 YYYY-MM-DD 형식으로 입력하면 자동 포함됩니다.
  • 필드가 비어 있으면 해당 항목은 JSON-LD에서 자동으로 제외되어 깔끔한 마크업을 유지합니다.
  • 생성된 코드를 복사한 뒤 구글 리치결과 테스트에서 즉시 검증하세요.
  • FAQ는 실제 페이지에 동일하게 표시되어야 검색엔진 가이드라인을 준수합니다.
사용설명서 & FAQ
  1. 기본 정보 영역에서 타입, 제목, 설명 등을 입력합니다.
  2. 별점 & 리뷰 데이터로 실제 평균 평점과 리뷰 수를 기입합니다.
  3. FAQ 항목을 추가하고 질문/답변을 작성합니다.
  4. 우측 JSON-LD 출력을 복사해 페이지 <head>에 삽입합니다.
  5. 검색엔진 테스트 툴로 검증 후 배포하면 리치결과 노출을 기대할 수 있습니다.
