도움말 & 팁
  • 피그마/스케치 등 디자인 툴에서 단위로 입력하려면 px 대신 pt를 사용해도 동일하게 적용됩니다.
  • 반응형 디자인에서는 브레이크포인트마다 컨테이너 폭을 변경해 저장 버튼으로 결과를 기록하세요.
  • 거터 값은 기본 단위의 배수로 맞추면 컬럼 폭이 소수점 없이 떨어져 개발 협업에 유리합니다.
  • 스케일 계수를 0.75, 1.25 등으로 조정하면 타입 스케일과 동일한 비율을 유지할 수 있습니다.
  • 복사 버튼을 사용하면 계산 로그가 자동으로 붙어 팀 문서에 공유하기 쉽습니다.
사용설명서 & FAQ
  1. 기본 단위와 스케일 계수를 설정합니다.
  2. 그리드 계산기에 현재 프로젝트 값을 입력해 컬럼 폭과 마진을 확인합니다.
  3. 빠른 라운딩으로 UI 여백을 가장 가까운 배수로 정리합니다.
  4. 스페이싱 표에서 필요한 단계를 클릭해 값과 메모를 복사합니다.
  5. FAQ 또는 팁을 참고해 디자인 시스템 문서에 반영합니다.
